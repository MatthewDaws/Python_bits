UK General election 2017
~~~~~~~~~~~~~~~~~~~~~~~~

1. `Look at the data.ipynb` - Explore the starting data, and process some data from the NOMIS website
2. `Previous Results.ipynb` - Use `requests` and `pandas` to crawl the web for the 2010 results.
3. `Some data manipulation.ipynb` - Messing with the data in Pandas, and using a random forest to predict the result from some demographic data (which works pretty well!)
4. `Make a crude prediction.ipynb` - Make a very silly prediction of the result.
